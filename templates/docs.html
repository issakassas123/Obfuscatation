API Documentation

GET /trusted_ips

Fetches the list of trusted IPs from the database.

Response

200 OK: Returns a JSON array of trusted IPs.
500 Internal Server Error: If there is an error fetching IPs.

GET /insert_key

Inserts a new key-value pair into the database. Requires API key for authentication.

Parameters

k1 (string): First part of the key. (required)
k2 (string): Second part of the key. (required)
v (string): Value to store. (required)

Response

200 OK: Returns a success message.
400 Bad Request: If any parameter is missing.
500 Internal Server Error: If there is an error inserting the data.

GET /get_key_db

Fetches the value for a specific key from the database. Requires API key for authentication.

Parameters

k1 (string): First part of the key. Use "all" to fetch all keys. (required)
k2 (string): Second part of the key. (required if k1 is not "all")

Response

200 OK: Returns the value for the specified key.
400 Bad Request: If any required parameter is missing.
500 Internal Server Error: If there is an error fetching the data.

GET /add_trusted_ip

Adds a new trusted IP to the database. Requires API key for authentication.

Parameters

ip (string): IP address to add. (required)

Response

200 OK: Returns a success message.
400 Bad Request: If IP is not provided.
500 Internal Server Error: If there is an error adding the IP.

GET /remove_ip

Removes a trusted IP from the database. Requires API key for authentication.

Parameters

ip (string): IP address to remove. (required)

Response

200 OK: Returns a success message.
400 Bad Request: If IP is not provided.
500 Internal Server Error: If there is an error removing the IP.

GET /update_config

Updates the configuration setting for blocking IPs. Requires API key for authentication.

Parameters

block_ips (string): New value for blocking IPs ("1" for true, "0" for false). (required)

Response

200 OK: Returns a success message with the new setting.
400 Bad Request: If the parameter is not provided or invalid.

GET /getAllIps

Fetches all trusted IPs from the database. Requires API key for authentication.

Response

200 OK: Returns a JSON array of trusted IPs.
500 Internal Server Error: If there is an error fetching IPs.

GET /getFirewallStatus

Fetches the current firewall status. Requires API key for authentication.

Response

200 OK: Returns the current status of the firewall (true/false).
